# ChillHub

**Your Android TV, Reimagined.**

A modern home screen launcher designed for simplicity, beauty, and putting you in control. Free forever, no ads, no tracking.

[![Google Play](https://img.shields.io/badge/Google%20Play-Download-green?logo=google-play)](https://play.google.com/store/apps/details?id=app.lumoslabs.chillhub)
[![GitHub Release](https://img.shields.io/github/v/release/AppleChillVibez/ChillHub?label=Sideload&logo=github)](https://github.com/AppleChillVibez/ChillHub/releases/latest)

## Features

- **Clean 6-column app grid** — your apps front and center
- **Dock row** — pin 6 favorite apps for quick access
- **Continue Watching shelf** — pulls from your installed apps
- **11 background themes** — Dark, Aurora, Cosmos, Forest, Waves, and more
- **Adjustable blur** — fine-tune the look (0–250dp)
- **Smart app sorting** — Most Used, Alphabetical, or Custom drag-to-reorder
- **App management** — hide, move, uninstall via long-press menu
- **Modern icon pack** — built-in overrides for a cohesive look
- **Clock widget** — time, date, or hidden
- **20 languages** supported

## Installation

### Google Play (recommended)

Download from the [Google Play Store](https://play.google.com/store/apps/details?id=app.lumoslabs.chillhub).

### Sideload (APK)

For devices not on the Play Store (e.g. Fire TV) or if you prefer manual installs:

1. Go to the [latest release](https://github.com/AppleChillVibez/ChillHub/releases/latest)
2. Download the `.apk` file
3. Install via one of these methods:

   **USB drive:**
   - Copy the APK to a USB drive
   - Plug into your TV and open with a file manager

   **ADB (from a computer):**
   ```bash
   adb connect <your-tv-ip>
   adb install ChillHub.apk
   ```

   **Send Files to TV (wireless):**
   - Install [Send Files to TV](https://play.google.com/store/apps/details?id=com.yablio.sendfilestotv) on both your phone and TV
   - Send the APK from your phone to the TV
   - Open and install on the TV

   **Downloader app (Fire TV):**
   - Install the [Downloader](https://www.amazon.com/dp/B01N0BP507) app on your Fire TV
   - Enter code **3908449**
   - Download and install

4. After installing, set ChillHub as your default launcher — see [hub.chillio.app/default](https://hub.chillio.app/default) for device-specific instructions
