# ChillHub - Bug Reports & Feature Requests

This is the public issue tracker for **ChillHub**. Use it to report bugs or request features.

## How to contribute

1. **Search first** — Check [existing issues](https://github.com/AppleChillVibez/ChillHub/issues) before creating a new one
2. **Vote, don't duplicate** — If your issue already exists, give it a 👍 reaction to help us prioritize
3. **Submit** — If it's new, [open an issue](https://github.com/AppleChillVibez/ChillHub/issues/new/choose) using one of the templates

## What happens next?

We review all submissions and use the 👍 count to help prioritize. We may ask follow-up questions in the comments.
